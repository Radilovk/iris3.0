ℹ️ Конфигурация на Cloudflare Worker (Wrangler CLI)

AI ключът и моделът се управляват чрез Wrangler. Изпълнете следните команди: 

# 1. Инсталирайте Wrangler (веднъж)
npm install -g wrangler

# 2. Влезте в Cloudflare
wrangler login

# 3. Създайте KV namespace и копирайте ID-тата в wrangler.toml
wrangler kv namespace create IRIS_KV

# 4. Задайте AI API ключ (поставете ключа когато бъдете попитани)
wrangler secret put AI_API_KEY

# 5. Редактирайте wrangler.toml ако искате друг модел или base URL
#    AI_MODEL   = "gpt-4o"
#    AI_BASE_URL = "https://api.openai.com/v1"

# 6. Деплойвайте Worker-а
wrangler deploy

# → Получавате URL от вида: https://iris-analyzer.<subdomain>.workers.dev
#   Поставете го по-горе в полето "URL на деплойнатия Worker".
